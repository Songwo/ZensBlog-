## 自定义配置

### 调整光点颜色
```css
/* src/app/globals.css */
.zen-floating-orb {
  background: radial-gradient(
    circle,
    rgba(255, 182, 213, 0.5), /* 调整这里 */
    transparent 70%
  );
}
```

### 调整光点大小
```css
.zen-orb-1 {
  width: 400px;  /* 默认 300px */
  height: 400px;
}
```

### 调整动画速度
```css
.zen-orb-1 {
  animation: zenOrbFloat1 6s; /* 默认 8s，更快 */
}
```

### 调整玻璃态模糊度
```css
.zen-glass-card {
  backdrop-filter: blur(20px); /* 默认 14px，更模糊 */
}
```

### 调整 hover 上浮距离
```css
.zen-glass-card:hover {
  transform: translateY(-12px); /* 默认 -8px，更高 */
}
```
